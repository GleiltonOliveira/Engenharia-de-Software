# SEGUE ABAIXO ORIENTAÇÕES SOBRE O BOTÃO FLUTUANTE DO WHATSAPP E
FEED DE IMAGENS DO INSTAGRAM #
-------------------------------------------------------------------------------------------------------

1° - Para configurar o número do botão do WhatsApp e o Feed de imagens
do Instagram acesse no painel administrativo da loja o ícone de 
engrenagem no canto superior direito  >>incluir código HTML >>
CONFIGURAÇÕES e abra esse bloco.

2° - Alterando o número do botão do WhatsApp :

# Onde está escrito var numWhats = 21972103517; altere somente o número
com o DDD sem espaços,traços ou pontos.

# Você também pode desabilitar o botão do WhatsApp,onde está escrito var
showButtom = true; basta substituir o true por false.


3° - Incluindo a sua conta do Intagram no carrossel do Feed de Imagens
do Instagram :

# Onde está escrito var instaToken =
'11050125846.1677ed0.590cd970a36d4211a90a18b1adf3eeeb';
substitua pelo seu token de acesso,você consegue e gerar e renovar (pois
o token expira com o tempo e tem que renovar) nesse site =>
https://instagram.pixelunion.net/ ,
o token de acesso tem que ficar entre
as aspas simples '' como no exemplo
('11050125846.1677ed0.590cd970a36d4211a90a18b1adf3eeeb').

# Onde está escrito var instaUserId = '11050125846'; substitua pelo seu
User ID
você consegue descobrir o seu user ID nesse site =>
https://codeofaninja.com/tools/find-instagram-user-id
o user ID tem que ficar entre as aspas simples '' como no exemplo
('11050125846').

# Você também pode desabilitar a seção de imagens do Instagram,onde está
escrito var instaFeedImgs = true;
basta substituir o true por false.


Não esqueça de salvar,na loja Integrada demora um tempo para que as 
alterações apareçam na loja
mas é só adicionar um produto ao carrinho de compras que já atualiza.
